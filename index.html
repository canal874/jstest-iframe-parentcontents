<iframe> parent/child playground

Usage guide:

  • This HTML document (parent) has many iframes (children) with different settings.
  • Push [button] => expected result (as of Jun.23, 2020) will be happen
  • Successful result will be shown on output area.
  • Catchable error will be shown on output area.
  • Uncaught error will be shown on console log of development tool. (Cross-origin error and Target-origin-not-match error in postMessage() cannot be try catch.)
  • Verified on Chrome and Firefox
  • Have fun!
